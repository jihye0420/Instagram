  > commit message

  ```
  CREATE - 기능 구현(한글)
  UPDATE - 코드 수정(한글)
  MERGE - 코드 병합(한글)
  FIX - 버그 수정(한글)
  RELEASE - 버전 배포(한글)
  DELETE - 기능 삭제(한글)
  DOCS - 문서 편집(한글)
  ```

  ex) `[UPDATE] 구현내용`
